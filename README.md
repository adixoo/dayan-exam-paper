   - First term (a) = $5$

   - Number of terms (n) = $20$

2. **Recall the formula for the sum of an arithmetic series:**

   $S_n = \dfrac{n}{2}(2a + (n-1)d)$

3. **Substitute the given values into the formula:**

   $S_{20} = \dfrac{20}{2}(2(5) + (20-1)(3))$

4. **Simplify the expression:**

   $S_{20} = 10(10 + 57)$
   $S_{20} = 10(67)$
   $S_{20} = 670$
